# 项目名称

报名表单

## 项目说明

制作了一个带基础校验的报名表单小项目，可以校验输入并展示提交结果。

## 字段设计

- `name`：姓名，用于存储报名者姓名
- `phone`：手机号，用于存储报名者手机号
- `email`：邮箱，用于存储报名者邮箱

## 核心功能

- 姓名非空校验
- 手机号非空 + 格式校验
- 邮箱非空 + 格式校验
- 校验失败时显示错误提示
- 校验通过后在页面展示提交结果
- 提交时页面不刷新

## 实现思路

1. 使用 `<form>` 标签包裹表单，监听 `submit` 事件
2. 调用 `e.preventDefault()` 阻止表单默认提交行为（防止页面刷新）
3. 校验时先清除之前的错误提示，然后逐个字段校验
4. 校验通过后在页面展示提交结果，并清空表单
5. 使用正则表达式进行手机号和邮箱格式校验

# 项目名称

书单管理

## 项目说明

制作了一个简单的书单管理小项目，可以添加书籍信息并展示列表。

## 字段设计

- `title`：书名，用于存储书籍名称
- `author`：作者，用于存储书籍作者
- `year`：年份，用于存储出版年份

## 核心功能

- 添加书籍到列表
- 删除指定书籍
- 动态渲染书籍列表
- 输入校验（不能为空）
- 可通过点击书籍切换书籍阅读完成情况

## 实现思路

1. 使用数组 `books` 存储所有书籍数据，每本书是一个包含 title、author、year 的对象
2. 定义 `render()` 函数，每次数据变化时清空列表容器，重新循环数组生成 `<li>` 元素并添加
3. 点击添加按钮时，获取输入框的值，校验非空后 `push` 到数组，然后调用 `render()` 重新渲染
4. 删除功能使用事件委托，监听列表的点击事件，通过 `dataset.index` 获取要删除的索引，用 `splice` 删除后重新渲染

# 书单管理

## 项目说明

一个图书收藏管理小应用，可以添加、删除书籍，标记已读/未读状态，数据保存在浏览器 localStorage 中，刷新不丢失。

## 字段设计

- **title**：书名，文本字段，显示在列表左侧
- **author**：作者，文本字段，显示在书名右边
- **year**：出版年份，数字字段，显示在作者右边
- **completed**：是否已读，布尔状态字段，默认为 false，点击书籍行切换状态，已读的书籍背景变深色

## 核心功能

- 添加书籍（书名、作者、年份三个输入框 + 添加按钮）
- 删除书籍（每条记录右侧有删除按钮）
- 切换已读/未读状态（点击书籍行即可切换，已读变深色背景）
- 统计显示（页面底部显示总数和已读数量）
- localStorage 持久化存储（刷新后数据不丢失）

## 实现思路

数据用 `books` 数组保存，每条数据是一个对象 `{ title, author, year, completed }`。所有增删改操作都修改数组，然后调用 `render()` 重新生成列表 DOM。`save()` 把数组 JSON 序列化后存入 localStorage，`load()` 在页面初始化时读取。

事件绑定采用事件委托思路：在 `render()` 中为每个 `li` 绑定点击切换状态，为每个删除按钮绑定删除事件。统计信息在 `render()` 末尾统一更新。

## 遇到的问题

1. **刷新后列表一片空白**：忘记在文件末尾调用 `render()`，导致 `load()` 读出了数据但没有渲染，加上 `render()` 后正常。

2. **删除书籍后底部统计不更新**：统计代码只写在"添加"按钮的事件里，删除和切换状态时不会触发。解决方法是把统计更新挪到 `render()` 函数内部，这样每次渲染列表都会刷新统计。

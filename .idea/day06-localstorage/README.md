# 项目名称

书单管理（本地存储版）

## 项目说明

制作了一个带本地存储功能的书单管理小项目，刷新页面后数据仍会保留。直接在day04的基础上进行的更改

## 字段设计

- `title`：书名，用于存储书籍名称
- `author`：作者，用于存储书籍作者
- `year`：年份，用于存储出版年份
- `completed`：完成状态，用于标记书籍是否已阅读

## 核心功能

- 添加书籍到列表
- 删除指定书籍
- 切换书籍完成状态
- 动态渲染书籍列表
- 数据持久化到localStorage
- 刷新页面后数据仍会保留

## 实现思路

1. 使用数组 `books` 存储所有书籍数据，每本书是一个包含 title、author、year、completed 的对象
2. 定义 `load_books()` 函数，从localStorage读取数据，如果没有则返回空数组
3. 定义 `save_books()` 函数，将数据序列化为JSON后保存到localStorage
4. 定义 `render()` 函数，每次数据变化时清空列表容器，重新循环数组生成 `<li>` 元素并添加
5. 添加、删除、切换状态时都调用 `save_books()` 保存数据
6. 页面加载时调用 `load_books()` 读取数据并渲染
